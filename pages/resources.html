Freenove 4WD Car and Remote Sketches

Below are two sketches: simplified_car_sketch.ino for the car,
and car_remote_sketch.ino for the handheld remote.

  • Download the Simplified Car Sketch
  • Download the Car Remote Sketch

Setting Up and Uploading the Sketches

  1. Download both sketches from the links above. You will end up with two files:
    • simplified_car_sketch.ino
    • car_remote_sketch.ino
  2. Organize your files:
    • Create a folder named simplified_car_sketch and place the simplified_car_sketch.ino file inside it.
    • Create a folder named car_remote_sketch and place the car_remote_sketch.ino file inside it.
    Note: The Arduino IDE requires the folder name to match the .ino filename.
  3. Open the Arduino IDE. If you do not have it installed, you can download it from Arduino’s official website.
  4. Update nRF24 addresses (if needed):
    • In both sketches, find a line like const byte addresses[6] = "Laser"; (or similar).
    • These addresses must match exactly between the car and the remote for communication to work.
    • If you have multiple cars, give each car/remote pair a unique address (e.g. "CarA", "CarB", etc.) so they don’t interfere with each other.
  5. Adjust speed reduction (optional) in the simplified_car_sketch.ino:
    • Look for a line float speed_reduction = 0.8;
    • Values closer to 1.0 allow for faster driving, while lower values like 0.6 slow it down.
    • Too low of a value might reduce torque on carpet or other high-friction surfaces.
  6. Upload the Car Sketch:
    1. Connect the Freenove Car’s main board to your computer with a USB cable.
    2. In the Arduino IDE, go to File > Open... and select simplified_car_sketch.ino.
    3. Under Tools > Board, select the board type (e.g. Arduino Uno).
    4. Under Tools > Port, choose the COM port or device corresponding to your board.
    5. Click Verify (the checkmark icon) to compile the code.
    6. Click Upload (the arrow icon) to send the code to your car board.
  7. Upload the Remote Sketch:
    1. Disconnect the car (if you have only one USB cable). Now connect the remote controller board (often another Arduino or similar) via USB.
    2. Open car_remote_sketch.ino in the Arduino IDE.
    3. Again, select the correct board and port in Tools > Board and Tools > Port.
    4. Make sure the addresses[] here matches the address in the car’s sketch exactly.
    5. Verify and Upload to the remote controller board.
  8. Test Your Setup:
    • Power the car using its battery pack (disconnect from USB if desired).
    • Power the remote with a battery or USB, depending on your setup.
    • When both are on and have matching addresses, moving the joystick or potentiometers on the remote should control the car.

Now your Freenove Car and its matching Remote Controller should be communicating via the nRF24 modules. Experiment with different speed reductions and addresses, but remember to keep them in sync on both the car and remote sketches!
